# 系统边界

## 目标

`omne-integrity-primitives` 提供无策略的完整性原语，让调用方复用 digest 解析、内容哈希和校验失败建模。

## 负责什么

- 解析 `sha256:<hex>` 形式的 digest。
- 解析调用方输入的原始 hex。
- 对字节流或 reader 计算 SHA-256。
- 对调用方提供的 JSON 记录计算链式 SHA-256。
- 产出结构化的 mismatch 错误。

## 不负责什么

- HTTP 下载。
- release 元数据抓取。
- 来源优先级和镜像策略。
- 安装编排、CLI 或产品错误码。

## 调用方边界

- 上层调用方负责提供字节来源。
- 上层调用方负责提供要参与链式哈希的序列化记录。
- 上层调用方负责决定什么时候校验、校验失败后如何处理。
