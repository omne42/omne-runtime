## ExecRequest

```rust
ExecRequest::new(program, args, cwd, required_isolation, workspace_root)
  .with_declared_mutation(bool)

ExecRequest::with_policy_default_isolation(program, args, cwd, policy_default_isolation, workspace_root)
  .with_declared_mutation(bool)
```

Fields:

- `program: OsString`
- `args: Vec<OsString>`
- `cwd: PathBuf`
- `required_isolation: IsolationLevel`
- `requested_isolation_source: RequestedIsolationSource`
- `workspace_root: PathBuf`
- `declared_mutation: bool`

`declared_mutation` is caller input. Use `with_policy_default_isolation(...)` when the caller is
intentionally delegating isolation selection to `GatewayPolicy::default_isolation`.

## RequestResolution

```rust
gateway.resolve_request(&request)
```

Fields:

- `program: OsString`
- `args: Vec<OsString>`
- `cwd: PathBuf`
- `workspace_root: PathBuf`
- `declared_mutation: bool`
- `input_required_isolation: Option<IsolationLevel>`
- `requested_isolation: IsolationLevel`
- `requested_isolation_source: RequestedIsolationSource`
- `requested_policy_meta: PolicyMetaV1`
- `policy_default_isolation: IsolationLevel`

`RequestResolution` is generated by `ExecGateway`, not constructed directly by downstream callers.
Use it when you need the gateway's canonical pre-execution request view, including explicit
policy-default provenance.

## ExecGateway Methods

- `supported_isolation()`
- `policy()`
- `capability_report()`
- `resolve_request(&ExecRequest)`
- `evaluate(&ExecRequest)`
- `execute(&ExecRequest)`
- `execute_status(&ExecRequest)`
- `execute_status_with_event(&ExecRequest)`
- `prepare_command(&ExecRequest, &mut Command)`

`execute()` is the primary API because it preserves `ExecEvent` and sandbox metadata.
`execute_status()` is a convenience helper that discards the event.

## CapabilityReport

- fields:
  - `supported_isolation`
  - `policy_default_isolation`
- helper methods:
  - `supported_policy_meta()`
  - `policy_default_policy_meta()`

These helper methods project the isolation fields into canonical `policy_meta::PolicyMetaV1`
fragments without changing gateway enforcement behavior.
